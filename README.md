# Cake Store


### Using
- Jsonwebtoken 
- Moment
- Multer
- Redux
- React-Redux
- Socket.io
- Dotenv
- Bcrypt
- React.js
- Nodemon
- Express
- MongoDB


```shell
$ npm install 
$ npm run dev
$ go to http://localhost:3000
```
```shell
  Add config.env to config folder
  In the file add PORT, MONGODB_URI
```
